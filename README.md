Sobre o projeto

Este sistema é uma lista de tarefas simples, criada utilizando somente HTML, CSS e JavaScript, sem qualquer framework proibido pela disciplina.
As tarefas criadas são armazenadas no LocalStorage, garantindo que permaneçam salvas mesmo após recarregar ou fechar o navegador.

Funcionalidades

Criar uma nova tarefa
Editar uma tarefa existente
Excluir uma tarefa
Marcar ou desmarcar uma tarefa como concluída
Salvar tudo automaticamente no LocalStorage

Tecnologias utilizadas

HTML5,CSS3,JavaScript e LocalStorage do navegador.

Como usar
Baixe ou clone o repositório.
Abra a pasta no Visual Studio Code.
Clique duas vezes no arquivo index.html para abrir no navegador.
O sistema já estará funcionando.
